🔬 Crossbar Array 3D Visualizer — Controls Guide

ہر button اور control کا مختصر جائزہ

📌 اوپر والے Header Buttons Header

Button کیا کام کرتا ہے
Iso Camera کو Isometric (ترچھا) view پر لے جاتا ہے — سب سے اچھا 3D نظارہ
Front Camera سامنے سے دکھاتا ہے — bottom bars صاف نظر آتے ہیں
Side Camera ایک طرف سے دکھاتا ہے — top bars کی direction واضح ہوتی ہے
Top اوپر سے نظارہ — پورا array grid کی طرح نظر آتا ہے
Auto-Rotate 3D scene خود بخود گھومنا شروع/بند کرتا ہے — presentation کے لیے
Theme ▼ 5 themes: Dark Blue/Neon · Pure Black · Deep Space · Light Gray · White (presentation)
Export ▼ PNG — Current View: screen کا screenshot save کرتا ہے
PNG — High-Res 2×: double resolution image (presentations کے لیے)
PNG — Transparent BG: شفاف background والی image (PowerPoint میں use کریں)
Reset All تمام settings واپس default پر — rows 4×4، سب colors اور sliders reset
Panel بائیں طرف کا control panel چھپاتا/دکھاتا ہے
Reset View (نیچے) Camera واپس Iso position پر — ادھر ادھر pan ہو جائے تو یہ button دباؤ

🖱️ Mouse Controls Mouse

Actionکیا ہوتا ہے
Left Click + Drag3D scene rotate کرتا ہے
Scroll WheelZoom in / Zoom out
Right Click + DragScene کو pan (کھسکانا) کرتا ہے
Hover on objectTooltip دکھاتا ہے — bar کا نام، row/column، layer

📐 Array Size Sidebar

Controlکیا کام کرتا ہے
Rows slider اوپر والے bars (Bit Lines) کی تعداد — 1 سے 16 تک
Columns slider نیچے والے bars (Word Lines) کی تعداد — 1 سے 16 تک
2×2 / 4×4 ... 16×16 Quick preset — ایک click میں array size بدل جاتا ہے

📏 Bar Dimensions Sidebar

Controlکیا کام کرتا ہے
Widthہر bar کی چوڑائی — چھوٹی value = پتلے bars
Heightہر bar کی موٹائی (اونچائی) — electrode thickness
Spacingbars کے درمیان فاصلہ — بڑی value = زیادہ پھیلا ہوا array
Overhangbars کا array کے باہر نکلنا — electrode overhang

⚡ Switching Layer Sidebar

Controlکیا کام کرتا ہے
Box (radio)ہر intersection پر مربع/چوکور switching element بناتا ہے
Cylinder (radio)ہر intersection پر گول cylindrical pillar بناتا ہے
Box SizeBox shape کا size (Box select ہو تو نظر آتا ہے)
DiameterCylinder کی چوڑائی (Cylinder select ہو تو نظر آتا ہے)
SW HeightSwitching element کی اونچائی — bars کے درمیان فاصلہ

📚 Stack Layers Sidebar

Controlکیا کام کرتا ہے
Countکتنے crossbar layers اوپر نیچے stack کریں — 1 سے 4 تک
Separationlayers کے درمیان عمودی فاصلہ

🎨 Colors Sidebar

Colorکیا رنگ بدلتا ہے
Bottom BarsWord Lines — نیچے والے horizontal electrodes
Top BarsBit Lines — اوپر والے عمودی electrodes
Switching Layerہر intersection پر موجود active material کا رنگ
Substrateنیچے کی base plate کا رنگ

👁️ Visibility Sidebar

Checkboxکیا چھپاتا/دکھاتا ہے
Substrateنیچے کی base plate
LabelsW0, W1... (Word Lines) اور B0, B1... (Bit Lines) labels
Gridفرش پر grid lines
AxesX (red) Y (green) Z (blue) محور — orientation دکھاتے ہیں
Intersectionsہر crosspoint پر چھوٹے سفید نشان
Shadowsobjects کے سائے
Opacity sliderتمام objects کی transparency — 1.0 = ٹھوس، 0.1 = تقریباً شفاف

⚙️ Presets Sidebar

Presetکیا لوڈ ہوتا ہے
Default 4×4معیاری 4×4 array — سب سے عام مثال
Dense 8×8گھنا 8×8 array — پتلے bars، کم spacing
Neural Net 8×48 rows × 4 cols — cylindrical pillars — neural network demonstration
Large 16×16سب سے بڑا 256 crosspoints والا array
3-Layer Stack3 crossbar layers اوپر نیچے — 3D integration demo
Cylindrical Pillars4×4 cylinder shape switching nodes کے ساتھ
Wide Spacing 4×4زیادہ فاصلے والا array — ہر element صاف نظر آتا ہے
